# 任务计划：停止生成功能

## 目标
实现流式生成过程中的中断功能，让用户可以随时停止 AI 回复生成。

## 成功标准
- [ ] 生成过程中显示"停止"按钮
- [ ] 点击停止按钮可立即中断流式输出
- [ ] 中断后保留已生成的部分内容
- [ ] 停止后可以继续发送新消息
- [ ] 支持 OpenAI 和 Claude 两个 API
