# Claude Code 命令速查指南（中文）

## 一、核心概念

Claude Code 通过「斜杠命令（/commands）」控制行为：

```
/init
/compact
/help
```

本质：用命令驱动 AI Agent 完成开发任务。

---

## 二、核心命令（必须掌握）

### 1. /init（项目初始化）

作用：

* 扫描当前代码仓库
* 生成 CLAUDE.md（项目记忆）
* 建立上下文

建议：

* 每个新项目必须执行一次

---

### 2. /compact（压缩上下文）

作用：

* 压缩历史对话
* 保留关键信息
* 降低 token 消耗

适用场景：

* 长时间开发
* 多步骤任务

---

### 3. /agents（多 Agent 管理）

作用：

* 创建多个 AI 子代理
* 支持任务分工协作

---

### 4. /clear（清空上下文）

作用：

* 清除所有历史对话
* 重新开始

---

### 5. /help（查看帮助）

作用：

* 查看所有可用命令

---

## 三、项目管理命令

### /add-dir

添加工作目录

### /memory

编辑 CLAUDE.md（项目记忆文件）

### /config

配置 Claude Code 设置

---

## 四、开发辅助命令

### /review

代码评审

### /pr_comments

查看 PR 评论

### /doctor

环境检测

### /status

查看当前状态

---

## 五、上下文与成本控制

### /cost

查看 token 消耗

### /usage

查看使用额度

---

## 六、高级能力

### /sandbox

安全执行环境（隔离运行代码）

### /mcp

外部工具连接（API / 插件）

---

## 七、推荐工作流

### 标准流程

```
/init
→ 描述任务
→ 执行任务
→ /compact
```

---

### 长任务流程

```
/init
→ 拆解任务
→ 执行
→ /compact
→ 继续执行
```

---

## 八、最佳实践

1. 必须先执行 /init
2. 长任务必须使用 /compact
3. 多任务用 /agents
4. 项目关键信息写入 CLAUDE.md

---

## 九、本质总结

Claude Code 本质是：

AI Agent + 命令行 + 项目记忆系统
